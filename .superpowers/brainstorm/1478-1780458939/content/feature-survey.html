SubTrack 功能盤點

勾選你有興趣新增或改善的項目（可多選）

🗂 訂閱列表

A

加回到期日欄位

目前只有計費週期，看不到實際哪天到期。可讓表格同時顯示到期日 + 計費週期，或取代其中一個。

B

部門 / 狀態 篩選下拉

目前只有文字搜尋，加入部門、狀態的 Select 篩選器，快速縮小範圍。

C

匯出 CSV

一鍵匯出目前篩選結果的訂閱列表，給財務/會計用。

💳 付款紀錄

D

篩選結果合計金額

查詢某日期區間後，表格底部顯示 TWD 合計與筆數。

E

付款紀錄匯出 CSV

匯出篩選後的付款紀錄，含服務名稱、日期、金額、幣別。

📊 Dashboard

F

趨勢圖時間範圍切換

目前固定顯示近 12 個月。加入 3M / 6M / 12M 切換按鈕。

G

下期預估費用卡片

根據每個訂閱的 billing_cycle + 最後付款日，推算「本月預計支出」並顯示在統計卡。

🔧 其他 UX

H

試用期 (trial_end_date) 標示

目前 trial_end_date 欄位在 UI 完全看不到。可在列表或詳情頁用標籤標出試用中的訂閱。

I

自動續訂 (auto_renew) 顯示

auto_renew 欄位有存但沒有在列表或詳情顯示。可在表格加一個小圖示或在詳情頁補上。
